🌐 SEACET Nexus – Smart Student Portal

SEACET Nexus is a modern, responsive student portal designed to provide students with quick access to essential campus information in one place. The project features a clean UI with both Dark Mode 🌙 and Light Mode ☀️ themes, offering a smooth and user-friendly experience.

Project Overview:
This is a frontend mini project built as part of our learning journey during the Destination Program conducted by Samara Simha Reddy. The goal was to apply modern web development concepts by creating a practical and visually appealing student portal.

Features:
Dark & ☀️ Light theme support
Interactive Dashboard
Planner section
Placements page
Notes section
Events & Clubs
Faculty information
Notifications & Search UI
Responsive and modern interface

Technologies Used:
HTML5
CSS3
JavaScript
Responsive Web Design


AI Usage:
This project was primarily developed manually. AI was used only to assist with a few UI ideas, content suggestions, and minor code improvements. The overall design, implementation, and integration were completed by us.

Learning Outcome:
Through this project, we gained hands-on experience in:

Building responsive web interfaces
Creating modern UI/UX designs
Implementing Dark & Light themes
Structuring a frontend project
Applying best practices in HTML, CSS, and JavaScript
