## ✨ Features

### 🏠 Home Page
- Dynamic greeting based on time of day
- Live clock & date display
- Weather widget (OpenWeather API)
- Random motivational quotes
- Quick access cards to all sections
- Notice board with latest updates
- Emergency contacts
- Animated statistics counters

### 📋 Academic Planner
- Add, edit, delete tasks (Assignments, Exams, Labs, Projects, Deadlines)
- Countdown timers for each task
- Filter by type and status
- Mark as completed
- Overdue detection
- All data stored in Local Storage

### 💼 Placement Hub
- 12+ company profiles with package, role, eligibility
- Search and filter companies
- Favorite companies (saved in Local Storage)
- Interview tips accordion
- Required skills tags

### 🔬 Internship Tracker
- Track applications: Applied → Interview → Offer
- Status chips & progress bars
- Add/edit/delete internships
- Wishlist feature
- Local Storage persistence

### 👨‍🏫 Faculty Directory
- Search faculty by name
- Filter by department
- Contact details (email, phone, cabin)
- Specialization info

### 🗺️ Campus Explorer
- Interactive SVG campus map
- Click buildings for information
- Facility details & working hours

### 📝 Student Notes
- Filter by semester & department
- Search notes
- Favorite & download functionality
- Subject-wise organization

### 🎭 College Clubs
- 8+ clubs with details
- Join/leave clubs (Local Storage)
- Upcoming events per club
- Category filtering

### 📅 Events
- Timeline design
- Filter by type & status
- Countdown for upcoming events
- Workshop, Hackathon, Seminar categories

### 👤 Student Profile
- Editable profile with photo, department, semester
- Skills with animated progress bars
- Certificates management
- CGPA tracking

### 📊 Dashboard
- Personalized overview
- Quick stats & actions
- Career roadmap cards
- Resume builder
- Coding zone
- Achievement badges
- Mini calendar

### ⚙️ Settings
- Dark mode toggle
- Accent color customization (6 colors)
- Font size adjustment
- Data export/import
- Storage management

## 🎨 Design Philosophy

SEACET Nexus follows a **premium design language** inspired by:

- 🍎 **Apple** — Clean typography, generous whitespace
- 🔷 **Google Material** — Card-based layouts, meaningful motion
- 🌊 **Glassmorphism** — Frosted glass effects, blur backgrounds
- ✨ **Micro-animations** — Smooth transitions, scroll reveals, hover effects

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Primary Blue | `#1a73e8` | Main brand color |
| Secondary Blue | `#4285f4` | Interactive elements |
| Accent Blue | `#8ab4f8` | Highlights |
| Success Green | `#34a853` | Positive states |
| Warning Yellow | `#fbbc04` | Caution states |
| Error Red | `#ea4335` | Error states |

## 🚀 Getting Started

1. **Clone or download** the project folder
2. **Open `index.html`** in any modern web browser
3. That's it! No build tools, no npm, no server needed.

### Recommended Browsers
- Google Chrome 90+
- Mozilla Firefox 88+
- Microsoft Edge 90+
- Safari 14+
